# Claude Code Instructions for Portfolio Website

## Blog Writing Workflow

**IMPORTANT:** When asked to write a blog post, follow this workflow:

### Step 1: Read the Blog Guide
- **Always read first:** `/home/zubair-ashfaque/GitHubProject/documentation/architect-blog-guide.md`
- Follow the tone specified in the guide
- Follow ALL instructions in the guide

### Step 2: Create First Version and Get Approval
- Draft the blog post following the guide's specifications
- Present the draft to the user for approval
- Make any requested changes based on feedback

### Step 3: Update Main Page First
- Update the main page: `/home/zubair-ashfaque/GitHubProject/05-portfolio-website/zubairashfaque.github.io/index.html`
- Add the new blog entry to "The Journal" section
- The journal entries are in the `journalEntries` array in the JavaScript section
- Place new entries at the top (most recent first)

### Step 4: Create the Blog Page
- Create the blog post HTML file in the `blog/` directory
- Follow the naming convention: `blog/topic-name.html`
- Match the styling and structure of existing blog posts

### Step 5: Commit and Push
- Stage both `index.html` and the new blog post file
- Create a descriptive commit message
- Do NOT include Claude attribution in commits (user preference)
- Push to the main branch
